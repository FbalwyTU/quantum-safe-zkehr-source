# Security and Privacy

This repository is a public research artifact. It must not contain production secrets, real patient data, or private deployment material.

Do not commit:

- IBM Quantum API tokens.
- Private keys, wallet files, SSH keys, or requester private keys.
- `.env` files.
- Mnemonic phrases or API credentials.
- Real patient records or clinical deployment data.

Cryptographic material generated by scripts in this repository is demonstration material only. Generated RSA keys, ML-KEM keys, AES record keys, wrapped keys, and IBM job metadata should be treated as local experiment artifacts and should not be reused in production.

IBM Quantum scripts read credentials from the process environment, for example `IBM_QUANTUM_TOKEN`. They should not print token values or store token values in result files.
